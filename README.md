# GenAI-Email-Classification
AI-powered multi-class email classification system using TF-IDF and Transformer embeddings.

# 📧 Automated Email Classification Using GenAI

## 🚀 Overview
This project builds a multi-class email classification system that categorizes emails into:

- Spam
- Promotions
- Support
- Personal

The system compares traditional TF-IDF + Logistic Regression with Transformer-based embeddings (GenAI).

---

## 🧠 Technologies Used
- Python
- Scikit-learn
- Hugging Face Transformers
- Sentence Transformers
- Pandas
- Matplotlib

---

## 📊 Models Implemented
1. TF-IDF + Logistic Regression
2. Transformer Embeddings + Logistic Regression

---

## 📈 Evaluation Metrics
- Confusion Matrix
- Macro F1-Score
- Accuracy

---

## 🔥 Results
Transformer-based embeddings improved semantic understanding and achieved higher F1-score compared to TF-IDF.

---

## 🎯 Real-Time Prediction
The system includes a function to classify new incoming emails dynamically.
